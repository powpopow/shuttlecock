How to set up (one time)
1
Create a new Google Sheet
Go to sheets.new and create a blank spreadsheet. Name it "Shuttlecock" or anything you like.
2
Open Apps Script
In your sheet: Extensions → Apps Script. Delete any existing code.
3
Paste the backend code
Download Code.gs (provided alongside this file) and paste its entire contents into the Apps Script editor. Click Save (Ctrl+S).
4
Run the setup function
In Apps Script, select function setup from the dropdown and click Run. Accept any permissions. This creates all 5 sheet tabs automatically.
5
Deploy as Web App
Click Deploy → New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and copy the URL.
6
Paste the URL above
Paste the deployment URL into the field at the top of this page and click Connect. That's it — your app is now linked to your Sheet.
Reset

Clear all locally cached data (does not affect Google Sheets).
